# LightBulb App

A simple React application demonstrating state management with useState and props.

## Overview

This app manages a lightbulb's on/off state using local component state and passes it to child components via props.

## Features

- Uses `useState` hook for local state management
- `BulbStatus` component displays the current bulb state
- `ToggleStatus` component toggles the bulb state via a callback prop

## Running the App

```bash
npm install
npm run dev
```
